# NETHAL Design Brief

## Product identity

Use the wordmark:

```text
NETHAL
```

Always uppercase.

Avoid:

- AI-looking typography
- generic tech gradients
- cyberpunk styling
- emoji-based UI
- playful consumer app language

## Visual direction

Android-first. Premium technical product.

References:

- UniFi Mobile
- Home Assistant Mobile
- Cloudflare
- Pixel UI
- Nothing OS
- Material 3 Expressive

## UI principles

- mobile-first
- dark theme by default
- technical but not cluttered
- one column layouts
- bottom navigation
- no drawer
- no desktop dashboard layout
- no fake hacker scanner
- no excessive charts
- no emoji icons
