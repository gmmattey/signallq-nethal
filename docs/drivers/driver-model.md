# Driver Model

## Philosophy

NETHAL is capability-first.

Applications should use capabilities instead of vendor-specific conditionals.

## Capability states

```text
AVAILABLE
UNAVAILABLE
REQUIRES_AUTH
EXPERIMENTAL
UNSAFE
UNKNOWN
```
